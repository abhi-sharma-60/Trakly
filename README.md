# Trakly 🚀

Trakly is an AI-powered competitive programming analytics and practice platform that aggregates coding profiles, analyzes performance, detects weak topics, and provides personalized problem recommendations with an integrated code execution sandbox.

## Key Features
- Unified LeetCode & Codeforces analytics
- Topic-wise and difficulty-wise insights
- AI-based weak topic detection
- Personalized practice recommendations
- Secure multi-language code execution sandbox

## Tech Stack
- Frontend: React, TypeScript, Tailwind CSS
- Backend: Node.js (NestJS-style architecture)
- Database: PostgreSQL
- Cache: Redis
- Code Execution: Docker-based sandbox
